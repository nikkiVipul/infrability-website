# Generated Image Prompts — Homepage Hero & Polish

Assets the redesign needs. Only the **hero orb** is essential; the rest are optional polish.
Drop finished files into `assets/img/` with the filenames noted, then enable the image swap
(see the commented block in `assets/styles.css` `.orb-stage`).

## Brand palette (paste into any generator)
- **Dark theme:** background midnight-navy `#05070d`; accents electric aqua-teal `#19e0d0`, azure blue `#2e9bff`, soft violet `#6f7bff`
- **Light theme:** background cool off-white `#f4f6fa`; accents deep teal `#0f9a8a`, azure `#1668c8`, violet `#4b54d6`
- Mood: premium, enterprise, governed, cinematic — **not** flashy/neon/cartoon. Blue→teal gradient with violet.
- A still is enough — motion is added in CSS/Canvas. Video/Lottie is optional.
